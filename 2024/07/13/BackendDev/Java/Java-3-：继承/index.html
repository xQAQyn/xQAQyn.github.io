字段访问权限

Java中的4类访问控制：

  • private：仅本类可见
  • protected：对本包和所有子类可见
  • public：对外部完全可见
  • 无修饰符：对本包可见

Object类

Object是所有类的超类，在显式的继承时，则是继承了Object类。在Java中只有基本类型不是类。

反射

反射是能够分析类能力的程序，主要用途有：

  • 在运行时分析类的能力
  • 在运行时检查对象，例如，编写一个适用于所有类的toString方法
  • 实现泛型数组操作代码
  • 利用Mthod对象实现类似于C++中函数指针的功能
