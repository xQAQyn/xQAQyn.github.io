此时理解迭代器的位置就相当关键，在迭代器初始化时，其被置于第一个元素之前，每次调用next就前进一步，并放回其跳过的元素，previous则是后退一步，同样返回跳过的元素。

迭代器的add方法会在迭代器之前创建新元素，并插入列表

而迭代器的remove和set方法，操作的则是上一次next或previous返回的元素

链表的迭代器需要小心并发错误，例如一个迭代器指向一个元素前面的位置，而另一个迭代器删除了这个元素，此时就会出现错误。为了避免出现并发修改异常，迭代器代码编写需要遵循以下原则：

  • 可以为一个集合关联多个只读迭代器
  • 可以在此基础上再关联一个能同时读写的得带起

数组列表ArrayList

ArrayList封装了一个动态再分配的对象数组，通过get和set随机访问元素

散列集HashSet

HashSet要求类型参数必须有hashCode方法，可以实现快速的找到元素，其本质为一个链表数组，不同哈希值的元素被放入不同的桶中。其用法与其它列表差不多，不再赘述

树集TreeSet

TreeSet是HashSet的改进版本，它是一个有序集合，插入时可以以任意顺序插入，遍历时会按排序后的顺序呈现。其底层实现是一个红黑树。

由于有序，所以需要一共对应类型的Comparator<E>

映射

Java类库为映射提供了两个通用实现：HashMap和TreeMap，都实现了Map接口

HashMap堆键进行了散列，而树映射则是根据键的顺序将元素组织为一个搜索树。两者的选择上来说，HashMap有更高的效率，而TreeMap可以实现有序的遍历。
