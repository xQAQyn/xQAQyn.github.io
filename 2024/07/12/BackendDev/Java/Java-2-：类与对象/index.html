Posted on 2024-07-12

面向对象的程序设计

类之间的关系：

  • 依赖：“use-a”关系，一个类的方法使用或操纵另一个类的对象
    • 依赖会导致类之间的耦合，现代软件工程认为应当尽量减少类之间的依赖
  • 聚合：“has-a”关系，一个类包含另一个类的一些对象
  • 继承：“is-a”关系，一个更特殊类与一个更一般的类的关系
